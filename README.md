### Books Store app
* Ruby on Rails
* PostgreSQL
* GrpahQL
* RSpec

## Steps to Run application
* git clone git@github.com:nileshkumar/books-store.git
* bundle install
* rails db:create
* rails db:migrate
* rails server
* rails app: run http://localhost:3000/
* graphQL api: run http://localhost:3000/graphql
